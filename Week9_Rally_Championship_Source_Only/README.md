# Rally Championship Manager (Week 9)

This project is a simple Java console application that simulates a rally championship system.

Features:
- Driver registration
- Rally car types (Gravel and Asphalt)
- Race result recording
- Championship standings and leader
- Championship statistics
- Car performance calculation

Technologies:
- Java
- Object-Oriented Programming
- Singleton pattern
- Interfaces and abstract classes

How to run:

javac *.java
java Main
